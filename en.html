How it works

Η Yumstar δημιουργεί εικονικούς σερβιτόρους σε κάθε γωνιά της επιχείρησης!

And how Yumstar works??

Οι πελάτες μπορούν πλεόν να παραγγείλουν από την κινητή τους συσκευή μέσα σε μια επιχείρηση εστίασης!
Στην επιχείρηση θα υπαρχούν Qr-codes που θα μπορούν οι πελάτες να σκανάρουν και να παραγγείλουν. Το προσωπικό της επιχείρησης θα έχει ώς ρόλο μόνο την παράδοση των προϊόντων!*

*Ο πελάτης μπορεί να επιλέξει εαν θέλει να παραγγείλει μέσο του προσωπικού ή της εφαρμογής.Η Yumstar δεν καταργεί τον σερβιτόρο.

Ηλεκτρονική Διαχείρηση

Η διαχείρηση των παραγγελιών και του καταλόγου της επιχείρησης γίνεται πιο εύκολη από ποτέ!

  • Μέσω της ηλεκτρονικής μας πλατφόρμας οι επιχειρήσεις μπορούν να διαμορφώσουν τους καταλόγους τους και να δούν τα στατιστικά στοιχεία τους.
  • O επιχειρηματίας μπορεί να παρακολουθήσει τα δεδομένα της επιχείρησης του σε οποιαδήποτε μέρος!

    • Ηλεκτρονική προβολή

    Η προβολη των τρέχουσων παραγγελιών μπορεί να γίνει σε οποιονδήποτε υπολογιστή ή Tablet!

  • Το προσωπικό του μπουφέ ή της κουζίνας μπορεί να διαχειρίζεται τις τρέχουσες παραγγελίες σε οθόνη.
  • Οι παραγγελίες μπορούν να φιλτράρονται ανά προιόν ή ανά τραπέζι.

    • POS Printer

    Δυνατότητα εκτύπωσης των παραγγελιών σε θρεμικούς εκτυπωτές.

    Η Yumstar would be soon compartible with Pos printers!

    Έτσι οι παραγγελίες μέσα από την εφαρμογή της Yumstar θα μπορούν ταυτόχρονα να προβάλονται σε κάποια οθόνη και να εκτυπώνονται!
